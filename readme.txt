== Description ==

Displays the Twitter-Images of all your friends as a mosaic in the sidebar of your blog. The plugin is widget ready and comes with many configuration options!

== Installation ==

1. Unpack the zipfile twittergrid-X.y.zip
1. Upload folder `twittergrid` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Place `<?php if(function_exists('twittergrid_display')) twittergrid_display(); ?>` in your template or use the sidebar widgets.
